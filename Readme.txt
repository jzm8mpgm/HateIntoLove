#hatetolove
#Twitter ASCII project

Plan

- User replies to any tweet with @TurnHateIn2Love
- The characters used in the previous tweet are captured
- These character are used to form one of a few ASCII images (heart/flower/dove)
- This new image is posted from @TurnHateIn2Love in reply
- All original tweets and new outputs are displayed on a website
